درباره من

من یک توسعه‌دهنده فرانت‌اند هستم که با اشتیاق به طراحی رابط کاربری زیبا، واکنش‌گرا و کاربرمحور علاقه‌مندم. مهارت اصلی من در HTML، CSS و JavaScript است و تجربه کار با فریم‌ورک‌هایی مانند React و Vue را نیز دارم. در طراحی رابط کاربری از ابزارهایی مانند Figma و Adobe XD استفاده می‌کنم تا تجربه‌ای حرفه‌ای برای کاربران ایجاد کنم. با استفاده از Tailwind CSS و Sass پروژه‌ها را سریع‌تر و با ساختاری بهتر توسعه می‌دهم. به اصول طراحی ریسپانسیو و Mobile-First کاملاً مسلط هستم. کار با Git و GitHub بخش مهمی از روند کاری من برای کنترل نسخه و همکاری تیمی است. تجربه‌ی خوبی در ارتباط با APIها و کار با RESTful API دارم. به دنبال یادگیری مداوم هستم و همیشه سعی می‌کنم خودم را با تکنولوژی‌های روز به‌روز نگه دارم. به دیزاین حساس هستم و جزئیات برایم اهمیت زیادی دارد. هدف من ساخت محصولاتی است که نه‌تنها زیبا بلکه سریع، قابل دسترس و کاربرپسند باشند.
